# 南極スライダー 👑🐧🦭

「南極スライダー」は、氷の上をツルツル滑るコマを操って遊ぶ、2人対戦型のブラウザ・ボードゲームです。 ペンギンチームとアザラシチームに分かれて、自分の「王冠」をいち早くゴールへ導きましょう！

## 🧊 ゲームの目的

自分のメインコマ（王冠👑をかぶったコマ）を、盤面中央にある**ゴール（🍧かき氷）**にぴったり止まらせたら勝ちです。

## 🎮 あそびかた

1.  **コマをえらぶ**
    
    *   自分のターンのとき、動かしたい自分のコマをタップ（クリック）して選択します。
        
2.  **すべる（移動する）**
    
    *   スマホ：動かしたい方向へ**スワイプ**します。
        
    *   PC：**矢印キー** または **W/A/S/Dキー** を押します。
        
    *   ⚠️ **注意:** 氷の上なので、**壁や他のコマにぶつかるまで止まれません！**
        
3.  **王冠をゴールへ**
    
    *   他のコマを壁がわりにうまく配置して、王冠コマをゴール（中央）に導きましょう！
        

## ✨ 特徴・機能

*   **クロスプラットフォーム対応:** PCでもスマホでもブラウザだけでサクサク動きます。
    
*   **リッチなアニメーション:** コマが滑る動きや、ぶつかった時の「むにゅっ」としたバウンドをCSSアニメーションで表現しています。
    
*   **ランダム配置モード (🎲):** いつもの配置に飽きたら、サイコロボタンを押してランダム配置からスタートできます。
